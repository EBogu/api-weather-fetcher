# API Weather Fetcher

## What it does
This function prompts the user to input multiple city names. The function then returns to the user in a string format, which city has the highest temperature, highest precipitation and highest humidity at that moment.

## Requirements
The installation of the requests library and the Flask library are the only prerequisites:
```
pip install requests

pip install flask
```

## How to run
Call the function in your terminal:
```
city_comp()
```
## Web interface
A flask web app is available. To run it:
visit `http://127.0.0.1:5000` in your browser.
Then type in a list city names separated by commas.
Press add city button. 
Results should appear.
